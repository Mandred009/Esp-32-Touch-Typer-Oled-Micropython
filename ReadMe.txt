ESP 32 MICROPYTHON TOUCH TYPER

Mandred Industries' Touch Typer Demo

1) Esp 32 has many touch capacitive pins.

2) In this project I have used 4 such pins as buttons.

3) These buttons can be named as left, right , clear , ok.

4) Left key is used to navigate the letters and move towards the start of the list.

5) Right Key is used to navigate the letters and move towards the end of the list.

6) Clear Key is used to clear the oled screen and bring writing cursor to the top left of the screen.

7) Ok is used to type the selected letter to the oled screen.
